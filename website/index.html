Features

Alles was ein moderner XMPP-Client können muss

OMEMO Verschlüsselung

End-to-End-Verschlüsselung für Nachrichten und Dateien. Ihre Kommunikation bleibt privat.

MUJI Multi-Party Calls

Video- und Audiokonferenzen mit mehreren Teilnehmern. Mesh-basiert und dezentral.

GTK4 & libadwaita

Moderne, native GNOME-Integration mit adaptivem Design und Dark Mode Support.

Dateiübertragung

HTTP File Upload, Jingle File Transfer und OMEMO-verschlüsselte Dateien.

Gruppenchats

Multi-User Chats (MUC) mit Moderation, privaten Räumen und persistentem Verlauf.

Message Archive

MAM (XEP-0313) Support für Nachrichtensynchronisation über alle Geräte hinweg.

  • Linux (GTK4 ≥ 4.14, libadwaita ≥ 1.5)
  • GStreamer 1.24+ (für Audio/Video)
  • libomemo-c (für OMEMO Verschlüsselung)
  • libnice (für Videocalls)

Über DinoX

DinoX ist ein moderner XMPP-Client basierend auf Dino, erweitert um professionelle Multi-Party Videocall-Funktionalität (MUJI) und zahlreiche Verbesserungen für die tägliche Nutzung.

Das Projekt fokussiert sich auf sichere Kommunikation, moderne Benutzeroberfläche und zuverlässige Gruppenkommunikation - perfekt für Teams, Familien und Communities die Wert auf Privatsphäre und Dezentralität legen.

Entwicklung

DinoX wird aktiv entwickelt und erweitert. Contributions, Bug Reports und Feature Requests sind auf GitHub willkommen.
