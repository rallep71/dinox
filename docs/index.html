Features

Everything a modern XMPP client needs - and more

OMEMO Encryption

End-to-end encryption for all messages and files. Double Ratchet protocol for maximum security.

MUJI Group Calls

Audio and video conferences with multiple participants. Mesh-based, decentralized and without a central server.

Modern GTK4 UI

Native GNOME integration with libadwaita design. Dark mode, responsive layout and touchscreen support.

Group Chats (MUC)

Multi-User Chats with moderation, private rooms, role management and persistent history.

File Transfer

HTTP File Upload, Jingle File Transfer and encrypted OMEMO media. Images, videos, voice messages.

Message Archive

MAM (XEP-0313) for message synchronization. Delete history, create backups and chat across devices.

System Tray

StatusNotifierItem support for background operation. Notifications even when the window is closed.

Extended Settings

Server configuration, backup function, color scheme selection and many additional options.

45 Languages

Fully translated into 45 languages - 100% coverage. From Arabic to Chinese, from German to Vietnamese.

About DinoX

DinoX is a modern XMPP client based on Dino, extended with professional multi-party video call functionality (MUJI) and numerous improvements for daily use.

The project focuses on secure communication, modern user interface and reliable group communication - perfect for teams, families and communities who value privacy and decentralization.

Why DinoX?

DinoX is based on the excellent Dino project and extends it with additional features: System Tray support, extended server settings, delete history, contact management and much more.
