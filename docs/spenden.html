Unterstützung, Zahlungsabwicklung und Hinweise

Worum es geht

DinoX ist ein Open-Source-Projekt. Wenn Sie die Entwicklung unterstützen möchten, können Sie freiwillig spenden. Spenden sind freiwillige Unterstützungen und beinhalten keinen Anspruch auf Support, Features oder Gegenleistungen.

Spendenwege

  • Ko-fi: ko-fi.com/rallep
  • GitHub Sponsors: github.com/sponsors/rallep71

Zahlungsabwicklung / Daten

Die Zahlungsabwicklung erfolgt über die jeweiligen Plattformen (Ko-fi bzw. GitHub). Dabei werden personenbezogene Daten (z. B. Name, Zahlungsdaten, E-Mail) durch den Plattformanbieter nach dessen Bedingungen verarbeitet. DinoX selbst verarbeitet Zahlungsdaten nicht direkt.
