Privacy Policy

1. Controller

The controller responsible for data processing on this website is the person/entity listed in the Imprint.

2. Hosting (GitHub Pages) / server logs

This website is hosted on GitHub Pages, a service provided by GitHub, Inc. When you access this website, GitHub (as hosting provider) may process technical connection data in server logs (e.g. IP address, date/time, requested URL, user agent) to deliver the website and for security purposes.

Legal basis (GDPR): Art. 6(1)(f) (legitimate interests, e.g. secure operation). For information about GitHub's processing, please refer to the GitHub Privacy Statement.

3. Local storage (theme preference)

This website stores your theme preference (light/dark) in your browser's local storage. This data stays on your device and is not transmitted to us.

Legal basis (GDPR): Art. 6(1)(f) (legitimate interests, providing a user-friendly website).

4. GitHub API request (optional)

The website may request the latest DinoX release information from GitHub (api.github.com) to display the newest version number. When this happens, your browser connects directly to GitHub and GitHub will receive technical connection data (e.g. your IP address). GitHub processes this data under its own responsibility. GitHub's privacy policy: GitHub Privacy Statement.

5. Contact

If you contact us (e.g. by email), we process the data you provide (e.g. your email address and the content of your message) to respond to your request.

Legal basis (GDPR): Art. 6(1)(b) (pre-contractual communication, where applicable) and/or Art. 6(1)(f) (legitimate interests).

6. Donations (third-party platforms)

If you donate via Ko-fi or GitHub Sponsors, the payment processing and any associated personal data processing are handled by those platforms. DinoX does not receive or process your payment details directly.

When you click a donation link/button, your browser connects to the respective provider (e.g. Ko-fi or GitHub) and transmits technical connection data (e.g. your IP address, user agent) as part of the HTTP request.

7. External links

This website contains links to external websites (e.g. GitHub, Ko-fi). When you follow those links, the respective provider processes data under its own responsibility.

8. External resources (fonts)

This website serves its fonts locally from the same domain. No fonts are loaded from Google Fonts.

8a. Translation service (Google Translate)

This website offers an optional language switcher powered by Google Translate. When you select a language other than English, a script from Google (translate.google.com) is loaded and your browser connects to Google servers. Google may set cookies (e.g. googtrans, NID) and receive technical connection data (IP address, user agent, page content).

Google processes this data under its own responsibility. For details, see the Google Privacy Policy.

Legal basis (GDPR): Art. 6(1)(a) (consent) — translation is only triggered by an active user action (clicking a language button).

9. Your rights

Depending on the applicable law (in particular the GDPR), you may have rights including access, rectification, erasure, restriction, data portability, and objection.

You also have the right to lodge a complaint with a supervisory authority.

10. Changes

We may update this privacy policy when the website or its processing activities change.
